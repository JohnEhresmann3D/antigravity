ASSET OVERVIEW
==============

All assets follow the satirical happy aesthetic with:
- Bright, vibrant colors (pastels and saturated hues)
- Thick black outlines (2-3 pixels)
- Exaggerated proportions
- Cheerful but slightly "off" vibe
- 90s cartoon style

BACKGROUNDS
===========

File: space_station_bg.png
Location: assets/backgrounds/space_station_bg.png

Description:
Parallax background showing cheerful space station interior with:
- Colorful cosmic space (pink, purple, cyan nebulas)
- Happy stars and planets
- Retro-futuristic station architecture
- Pastel colored metal panels
- Satirical signage and posters

Usage:
- Import as parallax background layers
- Separate into multiple layers for depth
- Scroll at different speeds for parallax effect

Recommended Layers:
1. Far background: Deep space (slowest scroll)
2. Mid background: Station exterior through windows (medium scroll)
3. Near background: Interior walls and details (faster scroll)

LEVEL DESIGN TIPS
=================

Color Coordination:
- Use pastel platforms (mint, pink, blue) for main paths
- Yellow/orange platforms for special areas
- Warning stripe platforms near hazards

Atmosphere:
- Place flowers as decorative elements
- Add propaganda posters for satirical humor
- Use vending machines and signs for world building
- Scatter furniture for lived-in feel

Hazard Placement:
- Laser barriers for horizontal obstacles
- Saw blades for moving hazards
- Electric barriers for vertical obstacles
- Steam vents for timing challenges or platforming aids

Parallax Background:
- Layer 1 (far): 0.2x scroll speed
- Layer 2 (mid): 0.5x scroll speed
- Layer 3 (near): 0.8x scroll speed
- Foreground platforms: 1.0x scroll speed

PERFORMANCE NOTES
=================

Mobile Optimization:
- Limit number of animated objects on screen
- Use object pooling for collectibles
- Batch similar sprites together
- Consider reducing animation FPS on lower-end devices

Recommended Limits (on screen at once):
- Animated flowers: 5-10
- Animated hazards: 3-5
- Static objects: 20-30
- Platform tiles: Unlimited (TileMap is efficient)


SATIRICAL ELEMENTS
==================

The "happy" aesthetic is intentionally over-the-top:
- Smiley faces on dangerous hazards (saw blades)
- Propaganda-style "BE HAPPY!" posters
- Mandatory fun signs
- Overly cheerful warning messages
- Bright colors in dangerous situations

This creates a satirical contrast between the cheerful visuals
and the actual platforming challenges, fitting the game's theme.
