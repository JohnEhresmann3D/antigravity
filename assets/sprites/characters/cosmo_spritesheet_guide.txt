ANIMATION BREAKDOWN
===================

ROW 1: IDLE (4 frames)
-----------------------
Frames: 1-4
Animation: Gentle breathing, slight helmet bob
Loop: Yes
Frame Rate: 8-10 FPS
Usage: Default standing state

Frame Details:
- Frame 1: Neutral pose
- Frame 2: Slight up movement (inhale)
- Frame 3: Peak of breath
- Frame 4: Down movement (exhale)


ROW 2: WALK (6 frames)
-----------------------
Frames: 1-6
Animation: Bouncy walking cycle, arms swinging
Loop: Yes
Frame Rate: 10-12 FPS
Usage: Horizontal movement (slow)

Frame Details:
- Frame 1: Left foot forward
- Frame 2: Mid-stride
- Frame 3: Right foot forward
- Frame 4: Mid-stride
- Frame 5: Left foot forward (opposite arm swing)
- Frame 6: Transition back to frame 1


ROW 3: RUN (6 frames)
----------------------
Frames: 1-6
Animation: Faster movement, exaggerated arm swing, leaning forward
Loop: Yes
Frame Rate: 12-15 FPS
Usage: Horizontal movement (fast)

Frame Details:
- More exaggerated than walk
- Leaning forward for speed
- Arms swing wider
- Faster cycle than walk


ROW 4: JUMP (4 frames)
-----------------------
Frames: 1-4
Animation: Wind-up crouch, launch, peak, descending
Loop: No
Frame Rate: 12-15 FPS
Usage: Jump action

Frame Details:
- Frame 1: Crouch (wind-up)
- Frame 2: Launch (pushing off)
- Frame 3: Peak (arms up, apex of jump)
- Frame 4: Descending (arms coming down)

Transition: Frame 4 → Fall animation if still airborne


ROW 5: FALL (3 frames)
-----------------------
Frames: 1-3
Animation: Arms flailing, worried expression
Loop: Yes (if falling continuously)
Frame Rate: 10-12 FPS
Usage: Falling state

Frame Details:
- Frame 1: Arms out, surprised
- Frame 2: Arms flailing up
- Frame 3: Arms flailing down
- Visor shows worried expression

Transition: Fall → Land when touching ground


ROW 6: LAND (3 frames)
-----------------------
Frames: 1-3
Animation: Impact squash, bounce, recovery
Loop: No
Frame Rate: 15 FPS
Usage: Landing from jump/fall

Frame Details:
- Frame 1: Squash (impact compression)
- Frame 2: Bounce (slight stretch up)
- Frame 3: Recovery (return to normal)

Transition: Land → Idle


ROW 7: GRAVITY FLIP (6 frames)
-------------------------------
Frames: 1-6
Animation: Spinning 180 degrees
Loop: No
Frame Rate: 12-15 FPS
Usage: Gravity flip ability activation

Frame Details:
- Frame 1: Start (normal orientation)
- Frame 2: 45° rotation
- Frame 3: 90° rotation (sideways)
- Frame 4: 135° rotation
- Frame 5: 180° rotation (upside down)
- Frame 6: Settle (upside down idle)

Notes:
- Can be reversed for flipping back
- Add particle effects in-game
- Visor shows surprise/excitement


ROW 8: DAMAGE (3 frames)
-------------------------
Frames: 1-3
Animation: Hit reaction, flash, knockback
Loop: No
Frame Rate: 15 FPS
Usage: Taking damage

Frame Details:
- Frame 1: Impact (recoil pose)
- Frame 2: Flash (white overlay or bright frame)
- Frame 3: Knockback (pushed back pose)

Notes:
- Add screen shake in-game
- Flash can be done with shader
- Visor shows pain expression

ANIMATION STATE MACHINE
========================

Recommended state transitions:

idle → walk (when moving)
idle → jump (when jump pressed)
walk → idle (when stopped)
walk → run (when speed increases)
walk → jump (when jump pressed)
run → walk (when speed decreases)
run → jump (when jump pressed)
jump → fall (when vertical velocity < 0)
fall → land (when touching ground)
land → idle (after land animation completes)
any → damage (when hit)
damage → idle (after damage animation)
any → gravity_flip (when ability activated)
gravity_flip → idle (after flip completes)

SCALING NOTES
=============

Current Size: 64x64 pixels per frame

For different screen sizes:
- Mobile: May need to scale up 2x or 3x
- PC: Can scale up to 4x for HD displays
- Use Godot's texture filtering: Nearest (for crisp pixels)

Collision Box:
- Recommended: 40x56 pixels (centered)
- Leaves room for animation overhang
- Adjust based on gameplay feel
